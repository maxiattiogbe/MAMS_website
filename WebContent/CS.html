Computer Science

This class is taught by Mrs. Tarrico. We begin the year by learning HTML and CSS we then use to design personal websites like this one. Our websites are supposed to function as our own personal and professional electronic portfolios. We had access to online resources such as W3Schools to help us with the website development.

Next, we learned fundamental concepts of the object-oriented programming language Java. The concepts we have learned so far include primitive data types, if and else statements, loops, boolean algebra, arrays, arraylists, implementing classes, and static methods. Next we are going to cover inheritance. Each unit starts with notes followed by exercises and usually also a lab or two.

In this class we write our Java, HTML, and CSS in the Eclipse integrated development environment (IDE).
